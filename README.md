# OneQue

OneQue is the working repository for the 모두의 창업 project.

## What Lives Here

- `app/`: lightweight MVP app.
- `.github/`: issue templates, PR template, CODEOWNERS, Dependabot, and CI.
- `docs/`: collaboration, review, triage, and GitHub setup guides.
- `make_pitch_deck.js`, `validate_pitch_deck.js`: pitch deck utility scripts.

Private business documents, PDFs, and draft office files should stay outside the public repository unless explicitly approved for publication.

## Local App

```powershell
cd app
npm run dev
```

The local server uses `server.mjs`.

## Operating Model

- GitHub issues track actionable work.
- Pull requests track reviewed changes.
- CI checks basic syntax and required collaboration files.
- Notion keeps meeting notes, decisions, requests, and dashboard context.
