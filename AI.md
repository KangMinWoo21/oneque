# 광고 대행 AX 플랫폼 프로젝트

## 1. 프로젝트 정체성

이 프로젝트는 광고주와 홍보 플랫폼이 더 직접적이고 투명하게 연결되는 광고 대행 AX 플랫폼을 만드는 창업 프로젝트다.

아직 서비스명은 정해지지 않았지만, 우리가 만들고자 하는 제품의 핵심은 분명하다.

광고 대행 과정에서 반복적으로 발생하는 비효율, 불투명한 추천, 높은 중간 비용을 AI와 데이터 기반 프로세스로 줄이고, 광고주가 광고 기획부터 매칭, 제작, 계약, 집행까지 한 플랫폼 안에서 처리할 수 있도록 돕는다.

초기 타깃은 청주 지역 소상공인이다. 학생회 및 지역 네트워크를 기반으로 작게 시작하고, 로컬 광고 시장에서 검증한 뒤 점차 더 넓은 시장으로 확장한다.

## 2. 창업을 결심한 이유

이 프로젝트는 실제 경험에서 출발했다.

동업자가 학생회 활동 중 플리마켓을 홍보하기 위해 광고 대행사를 통해 인플루언서 마케팅을 진행하면서, 기존 광고 대행 프로세스의 불편함과 비효율을 직접 겪었다.

당시 광고주는 광고 대행사를 거쳐 영상 결과물을 받고, 다시 피드백을 전달하고, 수정본을 기다리는 과정을 반복해야 했다. 의사소통은 여러 단계를 거치며 느려졌고, 광고주 입장에서는 왜 특정 인플루언서가 선택되었는지, 지불한 비용이 적절하게 쓰였는지 판단하기 어려웠다.

우리는 이 문제를 단순한 불편이 아니라, 로컬 광고 시장에서 반복되는 구조적 문제로 본다.

## 3. 기존 광고 대행 구조

기존 광고 대행 프로세스에는 보통 다음 네 주체가 존재한다.

1. 광고주
2. 광고 대행사
3. 광고 대행사로부터 외주를 받는 광고 영상 또는 포스터 제작사
4. 홍보 플랫폼: 인플루언서, 전광판 등

이 구조는 광고주가 직접 처리하기 어려운 일을 대신해 준다는 장점이 있지만, 여러 중간 단계를 만들고 각 단계마다 비용과 시간이 추가된다.

## 4. 우리가 발견한 문제

### 4.1 커뮤니케이션 피로와 느린 피드백

광고주는 광고 대행사를 통해 제작사와 간접적으로 소통한다. 결과물을 확인하고 피드백을 전달하는 과정이 길어지고, 수정 방향이 정확하게 전달되지 않거나 늦게 반영되는 문제가 생긴다.

광고주는 광고를 맡겼지만 실제 진행 상황을 실시간으로 파악하기 어렵고, 최종 결과물에 대한 통제감도 낮아진다.

### 4.2 홍보 플랫폼 선정의 불투명성

인플루언서나 홍보 채널이 어떤 기준으로 추천되는지 명확하지 않다. 많은 경우 팔로워 수처럼 단순한 지표에 의존하며, 실제 광고 성과에 영향을 줄 수 있는 요소들이 충분히 반영되지 않는다.

예를 들어 지역 적합도, 콘텐츠 카테고리, 참여율, 팔로워의 실제 반응, 광고주의 업종과의 궁합, 과거 캠페인 성과 같은 요소가 함께 고려되어야 한다.

### 4.3 높은 중간 비용

광고 대행사, 제작 외주, 플랫폼 연결 과정에 여러 인력이 개입하면서 중간 수수료가 높아진다. 그 결과 광고 단가가 올라가고, 특히 예산이 제한적인 소상공인은 광고 집행 자체를 부담스럽게 느낀다.

우리는 광고주가 꼭 필요한 도움은 받을 수 있어야 하지만, 불필요한 중간 단계 때문에 비용과 시간이 과도하게 늘어나는 구조는 바뀌어야 한다고 본다.

## 5. 우리가 만들고자 하는 해결책

우리는 AI와 추천 시스템을 활용해 광고 대행의 핵심 과정을 플랫폼화한다.

광고주는 플랫폼에 접속해 광고 목적, 업종, 지역, 예산, 원하는 홍보 방식, 제작 필요 여부 등을 입력한다. 플랫폼은 이를 바탕으로 적합한 홍보 플랫폼을 추천하고, 각 선택지별 예상 광고 효과를 함께 제공한다.

광고주와 홍보 플랫폼은 플랫폼 안에서 직접 매칭된다. 필요한 경우 AI가 계약서, 광고 영상, 포스터, 문구, 캠페인 기획안을 생성하고, 광고주가 직접 제작하기 어렵다면 검증된 외주 업체와 연결한다.

핵심은 광고주가 더 적은 마찰로 더 명확한 판단을 할 수 있게 만드는 것이다.

## 6. 목표 사용자

초기 목표 사용자는 청주 지역 소상공인이다.

이들은 광고의 필요성을 알고 있지만, 전문 광고 대행사를 이용하기에는 비용 부담이 크고, 직접 인플루언서를 찾거나 콘텐츠를 제작하기에는 시간과 경험이 부족할 수 있다.

초기에는 다음과 같은 사용자를 우선 고려한다.

- 플리마켓, 팝업, 지역 행사 운영자
- 카페, 음식점, 술집, 뷰티샵 등 지역 기반 매장
- 지역 대학가와 상권을 대상으로 홍보하려는 소상공인
- 소규모 예산으로 빠르게 광고 효과를 확인하고 싶은 광고주

## 7. 초기 제품 흐름

1. 광고주가 웹 또는 앱으로 플랫폼에 접속한다.
2. 광고하고 싶은 업종, 지역, 예산, 기간, 목표, 선호 홍보 방식을 선택한다.
3. 플랫폼은 조건에 맞는 홍보 플랫폼을 추천한다.
4. 각 추천 플랫폼별 예상 광고 효과, 추천 이유, 비용 범위, 적합도를 보여준다.
5. 광고주는 원하는 홍보 플랫폼과 매칭을 진행한다.
6. 광고에 필요한 계약서, 광고 문구, 영상, 포스터 등을 AI로 생성한다.
7. 직접 제작이 어렵거나 더 높은 완성도가 필요하면 외주 제작사와 연결한다.
8. 광고주, 홍보 플랫폼, 제작 외주 업체가 플랫폼 안에서 직접 계약하고 진행 상황을 관리한다.
9. 광고 집행 후 결과 데이터를 수집해 다음 추천과 성과 예측에 반영한다.

## 8. 핵심 기능 후보

### 8.1 광고 조건 입력

광고주는 광고 목적, 예산, 기간, 지역, 업종, 타깃 고객, 원하는 홍보 형식을 입력한다.

### 8.2 홍보 플랫폼 추천

협업 필터링, 콘텐츠 기반 추천, 지역 기반 필터링, 성과 예측 모델을 조합해 광고주에게 적합한 홍보 플랫폼을 추천한다.

초기에는 복잡한 AI 모델보다 명확한 규칙 기반 추천과 수작업 검증 데이터를 함께 활용할 수 있다. 이후 실제 캠페인 데이터가 쌓이면 추천 모델을 고도화한다.

### 8.3 예상 광고 효과 제공

각 홍보 플랫폼별 예상 도달 수, 예상 참여율, 예상 전환 가능성, 비용 대비 효율을 제공한다.

초기에는 예측값을 절대적인 성과 보장으로 표현하지 않고, 의사결정을 돕는 참고 지표로 제공한다.

### 8.4 AI 제작 도구

광고 문구, 포스터 초안, 영상 스크립트, 숏폼 콘티, 계약서 초안 등을 AI로 생성한다.

상용 LLM API와 이미지 또는 영상 생성 API를 활용하되, 법률 문서와 계약서는 사람이 검토할 수 있는 구조를 함께 마련한다.

### 8.5 직접 계약과 진행 관리

광고주와 홍보 플랫폼, 필요 시 제작 외주 업체가 플랫폼 안에서 직접 계약하고 진행 상황을 확인한다.

계약, 파일 전달, 피드백, 수정 요청, 일정 확인, 결제 상태 등을 한곳에서 관리하는 것이 목표다.

## 9. 제품 원칙

### 9.1 투명성

광고주는 왜 특정 홍보 플랫폼이 추천되었는지 이해할 수 있어야 한다. 단순히 "추천"만 보여주는 것이 아니라 추천 이유와 주요 판단 기준을 함께 제공한다.

### 9.2 비용 효율

불필요한 중간 단계를 줄여 광고 단가를 낮춘다. 다만 모든 중간 역할을 제거하는 것이 목표는 아니다. 필요한 전문성은 플랫폼 안에서 더 효율적으로 연결한다.

### 9.3 실행 가능성

초기 제품은 거대한 자동화 플랫폼이 아니라, 실제 청주 소상공인이 사용할 수 있는 작고 명확한 문제 해결 도구여야 한다.

### 9.4 로컬 검증

청주라는 구체적인 지역에서 시작해 실제 광고주, 인플루언서, 제작자 데이터를 확보한다. 로컬 시장에서의 신뢰와 반복 사용성을 먼저 검증한다.

### 9.5 사람과 AI의 협업

AI는 광고주를 대신해 모든 것을 결정하는 존재가 아니라, 광고주의 판단과 실행을 돕는 도구여야 한다. 추천, 제작, 계약 초안 생성은 AI가 돕고, 중요한 의사결정은 사용자가 이해하고 선택할 수 있어야 한다.

## 10. 사업 전략

초기 전략은 로컬 시장 집중이다.

청주 지역 소상공인과 학생회 네트워크, 지역 행사 경험을 기반으로 실제 광고 수요와 홍보 플랫폼 공급을 확보한다. 초기에는 완전 자동화보다 운영자가 일부 개입하는 방식으로 품질을 관리하고, 반복되는 패턴을 제품 기능으로 전환한다.

성공적으로 검증해야 할 것은 다음과 같다.

- 광고주가 기존 대행 방식보다 더 편하다고 느끼는가
- 추천된 홍보 플랫폼이 광고주에게 납득 가능한가
- 광고 비용이 실제로 줄어드는가
- 홍보 플랫폼과 제작 외주 업체도 플랫폼 참여를 원하게 되는가
- 캠페인 데이터가 쌓일수록 추천 품질이 개선되는가

## 11. 초기 검증 가설

우리는 다음 가설을 검증해야 한다.

1. 청주 지역 소상공인은 저렴하고 간편한 광고 집행 도구에 대한 수요가 있다.
2. 광고주는 인플루언서 추천 이유와 예상 효과가 명확하다면 플랫폼을 신뢰할 가능성이 높다.
3. 팔로워 수만 보는 추천보다 지역 적합도, 참여율, 업종 적합도를 함께 고려한 추천이 더 높은 만족도를 만든다.
4. AI로 생성한 광고 문구, 포스터 초안, 영상 기획안은 초기 광고 제작 비용과 시간을 줄일 수 있다.
5. 광고주와 홍보 플랫폼의 직접 계약 구조는 중간 비용을 줄이면서도 충분히 운영 가능하다.

## 12. 앞으로의 확장 방향

초기에는 청주 로컬 광고 시장에 집중한다. 이후 충분한 데이터와 운영 경험이 쌓이면 다른 지역으로 확장하고, 장기적으로는 업종과 지역에 구애받지 않는 광고 매칭 및 제작 플랫폼으로 발전시킨다.

확장 단계에서는 다음 방향을 고려한다.

- 지역별 인플루언서 및 홍보 채널 데이터베이스 구축
- 광고 성과 데이터 기반 추천 모델 고도화
- 광고주용 캠페인 대시보드 제공
- 홍보 플랫폼용 수익 관리 및 제안 관리 기능 제공
- 제작 외주 업체용 포트폴리오와 견적 관리 기능 제공
- 계약, 정산, 분쟁 대응 프로세스 표준화

## 13. AI가 이 프로젝트를 이해할 때 지켜야 할 기준

이 프로젝트를 돕는 AI는 다음 맥락을 항상 고려해야 한다.

- 이 프로젝트는 단순한 광고 제작 도구가 아니라 광고 대행 프로세스 전체를 재설계하려는 플랫폼이다.
- 초기 시장은 청주 지역 소상공인이며, 로컬 검증이 중요하다.
- 핵심 문제는 커뮤니케이션 피로, 추천의 불투명성, 높은 중간 비용이다.
- AI는 광고주와 홍보 플랫폼의 직접 연결을 돕는 보조자 역할을 해야 한다.
- 법률, 계약, 광고 성과 예측처럼 민감한 영역에서는 AI 결과를 확정적 사실처럼 표현하지 않아야 한다.
- 초기에는 완전 자동화보다 사람이 개입하는 운영형 MVP가 더 현실적일 수 있다.
- 데이터가 부족한 초기 단계에서는 추천 시스템을 규칙 기반, 수작업 검증, 간단한 점수화 모델부터 시작할 수 있다.
- 모든 기능은 "청주 소상공인이 실제로 광고를 더 쉽게 집행할 수 있는가"라는 질문으로 판단해야 한다.

## 14. 한 문장 정의

청주 소상공인이 광고 대행사를 거치지 않고도, AI 추천과 제작 도구를 통해 적합한 홍보 플랫폼을 찾고 광고를 실행할 수 있게 만드는 광고 대행 AX 플랫폼.

## 15. 현재 단계

현재 프로젝트는 아이디어와 문제 정의를 구체화하는 초기 단계다.

다음 단계는 서비스명 후보, MVP 범위, 초기 사용자 인터뷰 질문, 추천 기준 설계, 플랫폼 참여자별 핵심 화면 흐름을 정리하는 것이다.
